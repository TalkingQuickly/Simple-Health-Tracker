Simple health tracking system

Visit: http://sht.herokuapp.com for the live version

A simple Ruby on Rails application for generating a healthiness score based on a subjective assesment of key variables including; 
- Junk food eaten
- Alcohol consumed
- Sleep
- Exercise
- Vegetables eaten
- Smoking
